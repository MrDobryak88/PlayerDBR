# Music Player

Простой музыкальный плеер на JavaFX с базовым функционалом.

## Возможности

- Воспроизведение аудиофайлов (MP3, WAV)
- Пауза/возобновление воспроизведения
- Переключение треков (вперед/назад)
- Регулировка громкости
- Плейлист с возможностью добавления треков
- Отображение текущего трека и прогресса воспроизведения
- Сохранение/загрузка плейлиста

## Требования

- Java 11 или выше
- Maven 3.6+

## Установка и запуск

1. Клонируйте репозиторий:
   `ash
   git clone https://github.com/yourusername/musicplayer.git
   cd musicplayer
   `

2. Соберите проект:
   `ash
   mvn clean package
   `

3. Запустите приложение:
   `ash
   mvn javafx:run
   `

## Использование

1. Нажмите "Add Track" для добавления аудиофайлов в плейлист
2. Выберите трек из плейлиста для воспроизведения
3. Используйте кнопки управления для воспроизведения, паузы и переключения треков
4. Регулируйте громкость с помощью ползунка
5. Для сохранения плейлиста нажмите "Save Playlist"
6. Для загрузки сохраненного плейлиста нажмите "Load Playlist"
